# MAR-PD Contact Extractor v3.5

## Overview
Professional tool for extracting contact information (email/phone) from Facebook profiles. 
Designed **ONLY** for legitimate account recovery purposes.

## Features
- Extract registration email/phone from UID/Profile URL
- Multiple extraction methods
- Bangladesh phone number support
- JSON & TXT export
- Activity logging

# 1. Clone/download files
# 2. Run setup
